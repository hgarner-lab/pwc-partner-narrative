# PwC Partner Narrative Kits

A dependency-free prototype for turning approved PwC narratives into partner-ready posts, comments, emails and talking points.

## What is Included

- Three-section product flow: Narratives, Build Partner Kit and Proof Library.
- Approved narrative spine with core truth, tension, why it matters now, what PwC believes, proof, sector cuts, role cuts, moments and claims to avoid.
- Compact partner kit builder for sector, audience / role, client moment, output type and partner style.
- Separate Partner Kit output page with LinkedIn post starter, comment starter, client email starter, Q&A prep, proof to use, guardrails and make-it-yours prompts.
- Proof Library with approved PwC-hosted/public source proof and small proof-readiness scores.

## Run Locally

Open `index.html` directly, or serve the folder:

```sh
python3 -m http.server 4174
```

Then visit `http://localhost:4174`.

## Prototype Notes

The UI intentionally avoids official PwC marks and uses a text-based placeholder mark with a PwC-inspired warm palette. It does not implement public profile scraping or unapproved social monitoring. Partner perspective should be sourced from approved PwC-authored or PwC-hosted content.
